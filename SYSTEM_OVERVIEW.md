# Visión General del Sistema (System Overview)

**Propósito:** Definición general y visión del producto.

## Visión
Ser la plataforma líder en Moroleón y Uriangato para la difusión y venta segura de boletos digitales para eventos locales.

## Arquitectura
Sistema cliente-servidor basado en una aplicación Web (React SPA) y una API Backend (NestJS).

## Tecnologías
- **Frontend:** React, TypeScript, MUI
- **Backend:** NestJS, TypeScript
- **Base de Datos:** PostgreSQL
- **Infraestructura:** Docker, Nginx
